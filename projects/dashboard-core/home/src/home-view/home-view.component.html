Eclipse Dataspace Components
EDC is an open source framework hosted by the Eclipse Foundation for building secure, globally-scalable data-sharing services. EDC provides highly customizable components for creating control planes, data planes, decentralized identity systems, and federated data catalogs. Backed by leading companies and cloud providers, EDC gives developers the tools they need to deliver innovative solutions for data exchange networks.
